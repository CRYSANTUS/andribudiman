CSS Dasar

Tujuan Pembelajaran

Mahasiswa mampu memahami dasar-dasar CSS dan fungsinya dalam mengatur tampilan halaman web. Mahasiswa diharapkan dapat menggunakan selector dan properti CSS untuk mengatur warna, teks, dan tata letak, serta menerapkannya secara eksternal untuk menghasilkan tampilan website yang menarik dan konsisten.

Materi Pembelajaran

  • Pengertian CSS : CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan dan tata letak halaman web agar terlihat lebih menarik.
  • Cara menulis CSS (inline, internal, eksternal) : CSS dapat ditulis langsung di tag HTML (inline), di dalam <style> pada file HTML (internal), atau pada file terpisah .css (eksternal).
  • Selector CSS : Selector digunakan untuk memilih elemen HTML yang akan diberi gaya, seperti tag, class, atau id.
  • Pengaturan warna dan font : CSS digunakan untuk mengatur warna teks, background, jenis huruf, dan ukuran font.
  • Layout sederhana : CSS dapat mengatur tata letak halaman menggunakan properti seperti margin, padding, dan display.

Contoh Kode CSS

body {
    background-color: #f2f2f2;
    font-family: Arial;
}

h1 {
    color: blue;
}
