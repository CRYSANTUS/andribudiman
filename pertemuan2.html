HTML Dasar

Tujuan Pembelajaran

mahasiswa diharapkan mampu memahami konsep dan fungsi HyperText Markup Language (HTML) sebagai bahasa standar untuk membangun struktur halaman web. Mahasiswa mampu mengenali dan menggunakan struktur dasar dokumen HTML serta berbagai tag dan atribut untuk menyusun konten website. Selain itu, mahasiswa diharapkan dapat membuat halaman web statis sederhana menggunakan elemen HTML seperti heading, paragraf , tautan, dan daftar. Melalui pertemuan ini, mahasiswa diharapkan memiliki kemampuan dasar dalam membangun struktur website yang terorganisir dan mudah dipahami.

Materi Pembelajaran

  • Pengertian HTML : HTML (Hyper Text Markup Language) adalah bahasa markup yang digunakan untuk membuat struktur dan konten halaman web.
  • Struktur dasar dokumen HTML : Struktur dasar HTML terdiri dari <!DOCTYPE html>, tag <html>, <head>, dan <body> yang berfungsi sebagai kerangka halaman web.
  • Tag heading (h1â€“h6) : Tag heading digunakan untuk membuat judul dan subjudul pada halaman web, dengan ukuran dari h1 (terbesar) hingga h6 (terkecil).
  • Tag paragraf dan link : Tag <p> digunakan untuk membuat paragraf teks, sedangkan tag <a> digunakan untuk membuat tautan ke halaman atau sumber lain.
  • List (ul, ol, li) : List digunakan untuk menampilkan daftar item, dengan <ul> untuk daftar tidak berurutan, <ul> untuk daftar berurutan, dan <li> untuk setiap item daftar.

    • Contoh Struktur HTML

    <!DOCTYPE html>
<html> 
  <head>
    <title>Belajar</title>
  </head>
  <body>
    <h1>Hello World</h1>
  </body>
</html>
